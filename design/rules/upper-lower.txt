
# Day template

A day has a name
A day has exercises

# Determining the split

IF frequency is 3 days THEN
    IF user preference is Full body THEN
        Workout split is Full body OR
    ELSE
        Workout split is Push-pull-leg
ELSE IF frequency is 4 days OR 5 days THEN
    Workout split is Upper-Lower
ELSE IF frequency is 6 days THEN
    Workout split is Push-pull-leg

# Intializing the days

IF Workout split is Upper-Lower THEN
    Routine has a day named Upper1
    Upper1 focus is Upper
    Routine has a day named Lower1
    Lower1 focus is Lower
    Routine has a day named Upper2
    Upper2 focus is Upper
    Routine has a day named Lower2
    Lower2 focus is Lower

IF workout split is upper-lower AND frequency is 5 days THEN
    Routine has a day named Shoulder
    Shoulder focus is Shoulder

# Initialize template for each day (Hard-coded for now but can be improved)

IF Day focus is Upper THEN
    Day includes 2 chest exercises
    Day includes 2 back exercises
    Day includes 1 biceps exercise
    Day includes 1 tricep exercise
    IF frequency is 4 days THEN
        Day include 1 shoulder exercise

IF Day is Upper1 THEN
    Exercise 1 is chest
    Exercise 2 is back
    Exercise 3 is chest
    Exercise 4 is Back
ELSE IF Day is Upper2 THEN 
    Exercise 1 is back
    Exercise 2 is chest
    Exercise 3 is back
    Exercise 4 is chest
ELSE IF Day is Lower1 THEN
    Exercise 1 is quads
    Exercise 2 is hamstring
    Exercise 3 is glutes
    Exercise 4 is calves
ELSE IF Day is Lower2 THEN 
    Exercise 1 is hamstring
    Exercise 2 is quads
    Exercise 3 is glutes
    Exercise 4 is calves
ELSE IF Day is Shoulder THEN 
    All exercises are shoulder

IF Day is Upper THEN
    Exercise 5 is triceps
    Exercise 6 is biceps

IF Day is Lower THEN
    Exercise 3 is glutes
    Exercise 4 is calves

# Exercise matching

FOR EACH day:
    FOR EACH exercise of the day:
        IF exercise in fact base matches the target muscle group AND
            exercise in fact base is unassigned AND
            exercise in fact base has the highest priority among the unassigned AND
            exercise in fact base matches the user preferences AND
            exercise in fact base matches the user goal 
        THEN 
            exercise in fact base is assigned to day
