# Facts

A upper-lower program can be 4 days.
A upper-lower program can be 5 days.

# Rules

## Composition

IF program is upper-lower THEN
    IF frequency per week is 4 days THEN
        Day 1 is a upper day
        Day 2 is a leg day
        Day 3 is a upper day
        Day 4 is leg day
    ELSE IF frequency per week is 5 days THEN
        Day 1 is a upper day
        Day 2 is a leg day
        Day 3 is a upper day
        Day 4 is leg day
        Day 5 is a shoulder day
    END IF
ENDIF

IF program is upper-lower AND day is upper day THEN
    Day must have 2 chest exercises
    Day must have 2 back exercises
    Day must have 1 tricep exercise
    Day must have 1 bicep exercise
    IF frequency per week is 4 days THEN
        Day must have 1 shoulder exercise
    END IF 
END IF

IF program is upper-lower AND day is shoulder day THEN
    Day must have 1 front delt exercise
    Day must have 1 side delt exercise
    Day must have 1 rear delt exercise
END IF

IF program is upper-lower AND day is leg day THEN
    Day must have 1 quad exercise
    Day must have 1 hamstring exercise
    Day must have 1 glutes
    Day must have 1 calves
END IF

## Selection

IF program is upper-lower AND day is upper AND frequency is 4 times a week THEN
    IF priority is chest THEN 
        The day must start with a chest exercise
    ELSE IF priority is back THEN 
        The day must start with a back exercise
    ELSE IF priority is shoulder THEN 
        The day must start with a shoulder exercise
    ELSE
        One upper day must start with a chest exercise
        One upper day must start with a back exercise // ?? is this good
    END IF
END IF
